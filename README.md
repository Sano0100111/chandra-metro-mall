## ✨ Features

- 🎬 **Live Movies** — Now showing at INOX via TMDB API
- 📍 **Distance Calculator** — Real-time GPS distance from user
- 🗺️ **Google Maps** — One tap navigation to mall
- 🏢 **Floor Directory** — Interactive floor guide
- 🛍️ **Store Listings** — All 50+ shops and brands
- 🍔 **Food Court** — Barkavee Food Court section
- ⭐ **Feedback Form** — Star rating system
- 🤖 **AI Assistant** — Powered by NVIDIA Llama 4
- 📢 **Live Ticker** — Offers and announcements
- 📱 **Mobile First** — Fully responsive design
- 🌙 **Dark Luxury UI** — Premium red/dark theme
